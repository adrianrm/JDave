BDD framework for Java

Behaviour Driven Development (BDD) is a refinement for Test Driven Development (TDD). The actual process of doing BDD is same as in TDD but the vocabulary is different. It is believed that this change in vocabulary should be reflected in used tools too.

JDave is a BDD framework for Java. It is inspired by rspec and integrates JMock as mocking framework. We currently use JUnit 3 adapter to launch JDave specifications. This way we have IDE and build tool support from day one.
